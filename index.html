Welcome to BVN Web Design!

 Welcome to Blue Valley North's Web Design Class Website. This class is determined to help students begin their path of knowledge via HTML
(Hyper-Text-Markup-Language) and many more coding languages.

Why is Web Design Important?

 Web Design is an important class because it involves students learning HTML which is the base function of creating a website. The internet is a massive place today, and every day, more and more websites are being created. With the knowledge of HTML, students can build their own website and possibly pursue it as a life-long career.

What is this website?

 This website was made as a side-project for the teacher to help organize the class as there is more than one web design class, using this website will help you navigate around and learn more with the class at a faster pace.
